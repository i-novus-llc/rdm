# n2o-config-rdm

> N2O конфигурация для проекта RDM

## Install

```bash
npm install --save n2o-config-rdm
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'n2o-config-rdm'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
